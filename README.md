# NEEDY-GIRL-OVERDOSE-CHAT
 NEEDY GIRL OVERDOSE 主播女孩重度依赖 聊天软件 chatgpt AI ame



### 运行
在`demo.py`开头处添加openai key：

`os.environ["OPENAI_API_KEY"] = ""`

安装依赖：

`pip install -r requirements.txt`

运行：

`python demo.py`
